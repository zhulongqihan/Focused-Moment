## 🎮 核心游戏循环

```
1. 完成番茄钟（25分钟专注）
   ↓
2. 获得奖励
   - 普通: 100合成玉 + 300龙门币 + 50经验值
   - Boss: 200合成玉 + 500龙门币 + 100经验值
   ↓
3. 使用合成玉抽卡
   - 单抽: 600合成玉
   - 十连: 6000合成玉（保底5★+）
   ↓
4. 获得干员后进行养成
   - 升级: 消耗龙门币 + 经验值
   - 精英化: 消耗龙门币 + 经验值 + 芯片
   ↓
5. 继续完成番茄钟获取更多资源
```

## 💡 使用流程

### 首次使用
1. 双击 `Focused-Moment-v0.9.0-gacha-system.exe` 安装
2. 启动应用
3. 在"番茄战场"页面开始第一个番茄钟
4. 完成后自动获得100合成玉等奖励
5. 点击"🎲 寻访系统"进行首次抽卡
6. 点击"📋 干员收藏"查看获得的干员
7. 点击干员卡片进入详情页进行养成

### 日常使用
1. 每天完成6个番茄钟（每日目标）
2. 第6个番茄钟为Boss番茄钟（双倍奖励）
3. 积累合成玉进行十连抽（6000合成玉）
4. 使用龙门币和经验值升级干员
5. 达到等级上限后进行精英化

## 🔧 技术实现

### 架构
```
前端 (SvelteKit + TypeScript)
    ↕ Tauri IPC
后端 (Rust)
    ↕ SQLite
数据库 (本地存储)
```

### 核心模块
- `gacha/`: 抽卡系统（概率计算、单抽、十连）
- `operator/`: 干员系统（升级、精英化）
- `resources/`: 资源管理（货币、资源）
- `timer/`: 奖励系统（番茄钟奖励计算）

### 数据库表
- `operators`: 干员数据
- `currency`: 货币余额
- `resources`: 资源余额
- `gacha_history`: 抽卡历史
- `pity_counter`: 保底计数器
