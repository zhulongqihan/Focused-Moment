## 技术细节

### 图片功能
- 图片URL格式：`https://prts.wiki/images/{干员名}_1.png`
- 实现了图片懒加载（loading="lazy"）
- 图片加载失败时显示SVG占位符
- 图片容器使用 aspect-ratio 保持比例

### AI 功能
- 使用现有的通义千问 API 集成
- 生成100字以内的干员背景故事
- 需要在设置页面配置 API Key
- 优雅降级：未配置 API Key 时显示提示

### Boss 随机化
- 在 `startTimer()` 函数中检测 Boss 回合
- 调用 `selectRandomBoss()` 随机选择 Boss 名称
- 在计时器显示区域展示 Boss 名称

## 使用说明

1. 运行可执行文件启动应用
2. 完成番茄钟获得合成玉和资源
3. 前往寻访系统抽取干员
4. 在干员收藏页面查看干员图片
5. 在"赛博宠物"标签页选择干员并生成 AI 故事
6. Boss 回合时会随机显示明日方舟 Boss 名称

## 注意事项

- AI 功能需要在设置页面配置通义千问 API Key
- 干员图片来自 PRTS Wiki，需要网络连接
- 图片加载可能需要一些时间，已实现懒加载优化
