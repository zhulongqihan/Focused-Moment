# Release Notes - v0.9.0

## 🎮 明日方舟抽卡养成系统

发布日期: 2026-04-03

## 重大更新

本版本将原有的"量子仓鼠"宠物系统完全升级为明日方舟风格的干员抽卡养成系统，实现了完整的游戏化专注力培养闭环。

## 新增功能

### 🎲 抽卡系统（寻访系统）
- 完全复刻明日方舟的抽卡概率机制
- 单次寻访（600合成玉）和十连寻访（6000合成玉）
- 保底机制：50抽后概率递增，99抽必出6★
- 十连保底：至少一个5★或6★
- 实时显示保底计数器

### 👥 干员收藏系统
- 查看所有已获得的干员
- 按稀有度、职业筛选
- 按稀有度、等级、获得时间排序
- 重复干员自动提升潜能（最高6潜）
- 明日方舟风格的干员卡片展示

### 📈 干员养成系统
- **等级提升**: 消耗龙门币和经验值
  - Elite 0: 1-50级
  - Elite 1: 1-70级
  - Elite 2: 1-90级
- **精英化**: 消耗龙门币、经验值和芯片
  - 精英化后等级重置，等级上限提升
  - 需要达到当前等级上限才能精英化
- 干员详情页面，实时显示资源需求

### 💰 奖励系统（核心闭环）
- **普通番茄钟**: 100合成玉 + 300龙门币 + 50经验值
- **Boss番茄钟**: 200合成玉 + 500龙门币 + 100经验值
- **挑战完成**: 额外 +50合成玉 + 200龙门币
- 完成番茄钟后自动发放奖励并显示提示

### 🎨 界面优化
- 应用明日方舟主题样式
- 罗德岛装饰元素和源石纹理
- 稀有度颜色标识（金色6★、紫色5★等）
- 响应式设计，适配不同屏幕尺寸

## 技术改进

### 后端架构
- 新增Rust模块：
  - `gacha/`: 抽卡系统核心逻辑
  - `operator/`: 干员养成系统
  - `resources/`: 资源管理系统
  - `timer/rewards.rs`: 奖励计算系统
- SQLite数据库持久化存储
- 完整的错误处理和验证

### 测试覆盖
- ✅ 68个单元测试
- ✅ 9个集成测试
- ✅ 属性测试验证核心逻辑
- 测试覆盖率：核心功能100%

### 数据模型
- `Operator`: 干员数据（名称、稀有度、职业、等级、精英化、潜能）
- `Currency`: 货币（源石、合成玉、龙门币）
- `Resources`: 资源（经验值、芯片）
- `GachaResult`: 抽卡结果
- `PityCounter`: 保底计数器

## 游戏循环

```
完成番茄钟 → 获得奖励(合成玉/龙门币/经验值)
    ↓
使用合成玉抽卡 → 获得干员
    ↓
使用龙门币和经验值 → 升级干员
    ↓
使用芯片和资源 → 精英化干员
    ↓
继续完成番茄钟 → 获得更多资源...
```
