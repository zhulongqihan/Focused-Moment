### 手动推送步骤
1. 打开命令行（PowerShell 或 Git Bash）
2. 进入项目目录：
   ```bash
   cd "F:\Focused Moment"
   ```
3. 推送到 GitHub：
   ```bash
   git push origin main
   ```

### 如果推送失败
如果遇到网络问题，可以尝试：

1. **使用代理**（如果有）：
   ```bash
   git config --global http.proxy http://127.0.0.1:端口号
   git config --global https.proxy https://127.0.0.1:端口号
   ```

2. **使用 SSH 方式**：
   ```bash
   git remote set-url origin git@github.com:zhulongqihan/Focused-Moment.git
   git push origin main
   ```

3. **稍后重试**：
   网络恢复后再次运行 `git push origin main`
