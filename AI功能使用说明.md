# AI 功能使用说明

## 📖 概述

Focused Moment 集成了通义千问 AI，为你提供智能的专注分析和任务建议。AI 功能是**完全可选的**，不配置也不影响其他功能使用。

## 🔧 配置步骤

### 1. 获取 API Key

1. 访问 [阿里云百炼平台](https://dashscope.aliyun.com/)
2. 注册/登录账号
3. 进入控制台，创建 API Key
4. 复制你的 API Key

### 2. 在应用中配置

1. 打开 Focused Moment
2. 点击顶部导航栏的 **"设置"** 标签
3. 在 "AI 功能配置" 区域：
   - 粘贴你的 API Key
   - 点击 **"保存 API Key"** 按钮
   - 点击 **"测试连接"** 验证配置是否成功

## 🤖 AI 功能详解

### 1. 📊 每日总结（统计页面）

**位置**：统计星图 → 今日视图 → 右上角 "🤖 AI 总结" 按钮

**功能**：
- 分析你今天的专注数据
- 生成鼓励性的总结文字
- 包含：完成番茄钟数、专注时长、完成待办、Boss 胜场、挑战成功次数

**使用场景**：
- 每天结束时，回顾今日成果
- 获得 AI 的鼓励和反馈
- 了解自己的专注表现

**示例输出**：
```
📊 AI 总结：今天完成了 6 个番茄钟，专注 150 分钟，表现出色！
完成了 3 个待办事项，保持这个节奏，明天会更好！
```

### 2. 💡 任务优先级建议（待办页面）

**位置**：待办舱 → 右上角 "🤖 AI 建议" 按钮

**功能**：
- 分析你的待办列表
- 根据任务优先级和内容给出建议
- 推荐最应该优先完成的 3 个任务

**使用场景**：
- 任务太多不知道从哪开始
- 需要帮助确定优先级
- 想要更高效地安排工作

**示例输出**：
```
💡 AI 建议：建议优先完成：
1. 高优先级的"完成项目报告" - 截止日期临近
2. "回复重要邮件" - 影响团队进度
3. "整理线代笔记" - 为明天考试做准备
```

## ❓ 常见问题

### Q: 不配置 API Key 能用吗？
A: 可以！所有核心功能（番茄钟、待办、统计、宠物、成就）都不需要 API。AI 功能只是锦上添花。

### Q: API Key 安全吗？
A: API Key 存储在本地，不会上传到任何服务器。只在你点击 AI 按钮时才会调用通义千问 API。

### Q: AI 总结不准确怎么办？
A: AI 生成的内容仅供参考。你可以：
- 多点几次生成不同的总结
- 根据自己的实际情况调整
- 把它当作一种鼓励和反馈

### Q: 点击 AI 按钮没反应？
A: 请检查：
1. 是否已配置 API Key
2. 网络连接是否正常
3. API Key 是否有效（点击"测试连接"）
4. 查看底部提示信息了解具体错误

### Q: 想关闭 AI 功能？
A: 直接不使用即可，不点击 AI 按钮就不会调用 API，不会产生费用。

## 🎯 最佳实践
