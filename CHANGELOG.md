## [0.11.9] - 2026-04-05

### 修复
- 修复十连保底触发后保底计数状态不一致的问题，保证结果与状态同步。
- 修复单抽与十连抽在异常干员池场景下的崩溃风险（`expect`/`panic`），改为可传播错误。
- 修复干员持久化时 `class` 序列化失败会崩溃的问题，改为返回数据库错误。

### 变更
- 统一版本管理策略：以 `src-tauri/Cargo.toml` 作为版本权威来源。
- 设置页版本显示改为构建期注入，不再硬编码。
- 新增版本同步、版本校验与旧 exe 清理脚本，支持发布前自动检查。

### 构建与发布
- 加固 GitHub Actions 发布流程：增加 tag 格式与版本一致性校验。
- Windows 构建启用单线程并关闭增量编译，降低 rustc 在该环境的不稳定风险。
- 发布前清理历史构件并校验 exe 产物存在，避免空发布或混入旧文件。

## 版本说明

### 版本号规则
- **主版本号**：重大功能变更或不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型
- **新增**：新功能
- **变更**：对现有功能的变更
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **修复**：问题修复
- **安全**：安全相关的修复
