# Focused Moment v0.2.1 - 明日方舟主题增强版

## 🎨 新增功能 / New Features

### 明日方舟主题 (Arknights Theme)
- ✨ 增强罗德岛装饰角的可见度，采用蓝金渐变配色
- 🎯 添加源石纹理背景效果，增强古朴风格
- 💎 罗德岛风格边框，带有发光效果
- ◆ 标题添加罗德岛徽章动画
- 🌟 所有面板采用明日方舟配色方案

### 计时器模式 (Timer Modes)
- ⏱️ 倒计时模式：传统番茄钟，从设定时间倒数至0
- ⏲️ 正向计时模式：从0开始正向计时，适合自由专注

### AI 功能基础设施 (AI Infrastructure)
- 🤖 集成通义千问 API 支持
- 🔌 可插拔设计：无需 API Key 也能正常使用
- ⚙️ 配置管理系统，支持本地存储

## 🎯 优化改进 / Improvements

- 移除浮动小部件按钮，简化界面
- 增强视觉装饰的对比度和可见性
- 优化罗德岛主题色彩搭配
- 改进计时器控制逻辑

## 📦 下载 / Download

### Windows
- **可执行文件**: `Focused-Moment-v0.2.0.exe` (直接运行，无需安装)
- **安装程序**: `Focused Moment_0.1.0_x64-setup.exe` (NSIS 安装包)
- **MSI 安装包**: `Focused Moment_0.1.0_x64_en-US.msi`

## 🚀 使用说明 / Usage

1. 下载 `Focused-Moment-v0.2.0.exe`
2. 双击运行即可使用
3. 所有数据本地存储，无需联网

### AI 功能配置（可选）
- 如需使用 AI 总结功能，请在设置中配置通义千问 API Key
- 不配置 API Key 不影响其他功能使用

## 🎮 明日方舟元素说明

本版本融入了《明日方舟》的视觉设计元素：
- **罗德岛蓝** (#0098DC)：主色调，代表专注和理性
- **源石金** (#FFB800)：强调色，代表成就和奖励
- **装饰角**：模仿游戏中的 UI 框架设计
- **源石纹理**：交叉斜纹背景，增添科技感
