# Focused Moment v0.9.0 - 明日方舟抽卡养成系统

## 🎮 项目简介

Focused Moment 是一款结合番茄钟专注力训练和明日方舟风格抽卡养成的桌面应用。通过完成番茄钟获得奖励，使用奖励进行抽卡和干员养成，形成完整的游戏化专注力培养闭环。

## 📦 快速开始

### 下载和安装
1. 下载 `Focused-Moment-v0.9.0-gacha-system.exe`（4.1MB）
2. 双击运行安装程序
3. 启动应用开始使用

### 首次使用
1. 在"番茄战场"页面开始一个25分钟的番茄钟
2. 完成后自动获得 100合成玉 + 300龙门币 + 50经验值
3. 点击"🎲 寻访系统"使用合成玉进行抽卡
4. 点击"📋 干员收藏"查看和养成获得的干员

## ✨ 核心功能

### 🍅 番茄钟系统
- 工作/休息模式切换
- 倒计时/正向计时
- Boss番茄钟（达到每日目标时双倍奖励）
- 作战委托挑战系统

### 🎲 抽卡系统
- **单次寻访**: 600合成玉
- **十连寻访**: 6000合成玉，保底5★+
- **概率机制**: 完全复刻明日方舟
  - 6★: 2% (保底后递增)
  - 5★: 8%
  - 4★: 50%
  - 3★: 40%
- **保底**: 50抽后递增，99抽必出6★

### 👥 干员养成
- **升级**: 消耗龙门币和经验值
  - Elite 0: 1-50级
  - Elite 1: 1-70级
  - Elite 2: 1-90级
- **精英化**: 消耗龙门币、经验值和芯片
  - 精英化后等级重置，上限提升

### 💰 奖励系统
- **普通番茄钟**: 100合成玉 + 300龙门币 + 50经验值
- **Boss番茄钟**: 200合成玉 + 500龙门币 + 100经验值
- **挑战完成**: 额外 +50合成玉 + 200龙门币

## 🎯 游戏循环

```
完成番茄钟 → 获得奖励 → 抽卡获得干员 → 养成干员 → 继续番茄钟
```
