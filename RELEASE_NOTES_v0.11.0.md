# Focused Moment v0.11.0 发布说明

发布日期：2026-04-04

## 🎯 本次更新概述

本次更新重构了首页番茄钟系统，移除了Boss战机制，改为更平衡的线性奖励系统，并优化了计时模式的用户体验。

---

## ✨ 新功能

### 1. 线性奖励系统
- 移除Boss战机制，所有番茄钟奖励一致
- 按实际专注时长计算奖励：
  - 每分钟获得：4 合成玉、10 龙门币、2 经验值
  - 25分钟标准番茄钟：100 合成玉、250 龙门币、50 经验值
  - 支持自定义时长，奖励按比例计算

### 2. 优化计时模式切换
- 番茄钟模式（倒计时）和正向计时模式切换更清晰
- 运行时禁用模式切换，避免误操作
- 按钮文案优化：⏱️ 番茄钟 / ⏲️ 正向计时

### 3. 改进按钮逻辑
- 将"跳过"按钮改为"完成/结束"按钮
- 正向计时模式：显示"完成"，手动结束并获得奖励
- 倒计时模式：显示"结束"，提前结束当前会话
- 按钮状态根据计时器运行状态智能启用/禁用

## 🔧 改进

### 作战委托系统
- 统一所有委托奖励：+30 合成玉 +100 龙门币

- 简化委托描述，更易理解

### 用户界面
- 移除"Boss回合"显示，界面更简洁
- 优化按钮布局和文案
- 改进计时模式切换按钮样式

---

## 🐛 修复

- 修复Boss战机制导致的奖励不平衡问题
- 修复正向计时模式下按钮文案不准确的问题
- 修复计时器运行时仍可切换模式的问题

## 📊 奖励对比

### 旧系统（v0.10.0）
- 普通番茄钟：100 合成玉、300 龙门币、50 经验值
- Boss番茄钟：200 合成玉、500 龙门币、100 经验值
- 挑战奖励：+50 合成玉、+200 龙门币

### 新系统（v0.11.0）
- 标准番茄钟（25分钟）：100 合成玉、250 龙门币、50 经验值
- 长番茄钟（50分钟）：200 合成玉、500 龙门币、100 经验值
- 挑战奖励：+30 合成玉、+100 龙门币
- 支持任意时长，奖励按比例计算

## 🎮 使用建议

1. **标准番茄钟**：设置25分钟，适合大多数专注场景
2. **短番茄钟**：设置15-20分钟，适合快速任务
3. **长番茄钟**：设置40-50分钟，适合深度工作
4. **正向计时**：不确定任务时长时使用，完成后手动结束
5. **作战委托**：每次专注都接取委托，获得额外奖励
