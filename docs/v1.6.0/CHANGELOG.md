# v1.6.0 今日专注星图版

## 本版目标
- 把当前未发版的 `v1.6.0` 收成可继续演进的稳定基线。
- 围绕“今日专注星图”完成前端视觉增强和布局安全收尾。
- 清理已经脱离当前主线的历史残留，减少版本信息和功能叙事漂移。

## 主要改动
- 焦点页新增“今日专注星图”，基于当前计时状态、今日记录和今日待办展示当天节奏概览。
- 调整焦点页 hero / atlas / 计时面板布局，避免新视觉层遮挡按钮、文本或核心操作区。
- 统一前后端版本信息，运行时版本和里程碑对齐到 `v1.6.0` 与“今日专注星图版”。
- 移除未接线的 CSV 导出后端残留，避免继续保留半接线功能边界。
- 删除空壳 `TaskWidget.tsx` 与未使用的 `.widget-*` 样式。
- 恢复 `tsconfig.json` 对主前端目录的正常类型检查覆盖。
- README 补充“今日专注星图”能力说明，使文档叙事与当前主线一致。

## 验收结果
- 已通过 `pnpm build`。
- 已通过 `cargo check --manifest-path src-tauri/Cargo.toml`。

## 回归重点
- 焦点页空状态、运行中、休息中、恢复会话下的星图和操作区保持稳定。
- 窄窗下不出现中文竖排、重叠、遮挡和操作区跳位。
- 开发者页版本信息与运行时版本一致。
- 备份元数据中的 `appVersion` 与当前版本一致。
- 待办、复盘、备份恢复等原有工作流不受本轮影响。
