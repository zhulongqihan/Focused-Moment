# Focused Moment v1.6.0

## 本版重点

- 收拢主界面信息层级，让界面更像日常使用的桌面工具，而不是测试面板
- 延续 `v1.6.0` 的专注驾驶舱与今日专注星图主线
- 保留备份、显示模式和版本信息，但把内部说明尽量移出主界面

## 这次调整了什么

- 侧边导航改得更克制，主视图切换不再堆叠说明性摘要
- 专注页保留开始计时、模式切换、任务关联和今日星图，减少引导型文案
- 待办页和复盘页收起大段解释，让任务、筛选和趋势本身更先被看到
- `开发者信息` 视图调整为更用户向的 `设置与数据` 入口
- 显示模式、备份与版本信息继续保留，但内部模块路线和实现说明转移到文档维护

## 仍然保留的能力

- 正向计时与番茄钟
- 任务创建、编辑、删除、完成与专注关联
- 按范围复盘、趋势查看、记录删除与范围清理
- 本地完整备份导出与恢复
- 本地存储、异常恢复、单实例、托盘常驻

## 文档说明

从这一版开始，更多设计背景、实现说明和版本恢复分析会优先保存在仓库文档中，而不是继续放在应用主界面上。
