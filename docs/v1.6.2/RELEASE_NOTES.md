# Focused Moment v1.6.2

## 本版重点

- 修正 GitHub Release 发布链路里的中文编码风险
- 统一脚本侧的 UTF-8 输出，减少发布校验时的乱码和噪音感
- 延续版本同步纪律，把这次发布流程修正独立收成一个补丁版本

## 这次更新了什么

- `publish-release.ps1` 改为优先使用 `--notes-file` 发布说明，不再把整段中文说明直接塞进命令行参数
- 发布脚本在执行前会主动切到 UTF-8 输入输出编码，降低 PowerShell 与 `gh` 混用时的中文乱码概率
- `release-ship.ps1` 同步补上 UTF-8 控制台初始化，让整条发布链路的终端输出更稳定
- 版本信息统一更新到 `1.6.2`

## 核心能力仍然保持

- 正向计时与番茄钟
- 任务创建、编辑、完成和专注关联
- 按范围复盘、趋势查看和记录清理
- 本地备份导出与恢复
- 本地存储、异常恢复、托盘常驻、单实例

## 说明

`v1.6.2` 不是新的功能版本，而是对发布链路与版本纪律的一次补强，目的是让后续每次发版更稳定、更容易核对。
