# Focused Moment

Focused Moment 是一款面向 Windows 的轻量效率工具，把专注计时、待办清单和数据复盘放进同一个安静、克制的桌面界面里。

它适合这样的使用方式：
- 开始一轮正向计时或番茄钟
- 把这轮专注和一个待办关联起来
- 在结束后自动留下记录
- 回到数据页看看自己最近的节奏

## 你可以用它做什么

- `专注计时`
  支持正向计时和番茄钟，两种方式都可以记录正在做的事情。
- `待办管理`
  可以添加任务、设置日期、开始时间和重要程度，并在专注时直接关联；也可以按关键字搜索、按状态筛选、按时间或重要程度排序。
- `数据复盘`
  可以查看累计时长、专注次数、最近趋势和按日节奏，也可以按时间范围筛选、删除测试记录或导出 CSV。
- `后台常驻`
  点击右上角 `隐藏` 后，应用会驻留在系统托盘里继续运行；点击 `退出` 才会完全关闭。
- `单例运行`
  同一时间只保留一个应用实例；如果你再次双击程序，会直接回到已经打开的那一份。

## 日常使用方式

### 1. 开始专注
- 打开 `开始专注`
- 输入这次要做的事情
- 选择 `正向计时` 或 `番茄钟`
- 点击 `开始计时`

### 2. 管理今天的待办
- 打开 `管理待办`
- 添加今天要做的任务
- 给任务设置日期、时间和重要程度
- 在专注页面里可以直接把一轮专注关联到某个任务

### 3. 查看最近状态
- 打开 `查看复盘`
- 看总专注时长、最近趋势和每天的沉淀节奏
- 可以切换 `今天 / 近 7 天 / 近 30 天 / 全部 / 自定义`
- 如果只是测试数据，可以删除单条记录、清理当前范围，或者直接清空本地数据重新开始
- 如果你想把记录拿出去分析，也可以导出当前范围的 `CSV`

### 4. 查看内部信息
- 打开 `开发者信息`
- 这里会集中显示版本号、当前状态、运行方式和后续扩展预留
- 正常日常使用时可以完全忽略这一页

## 数据与隐私

- 当前所有任务、专注记录和统计数据都默认保存在本地
- 不依赖在线账号
- 不需要联网才能完成日常使用

## Windows 使用方式

如果你已经拿到构建好的程序，根目录里通常会看到这些文件：
- `Focused Moment.exe`
- `Focused Moment vX.Y.Z.exe`
- `Focused Moment Setup.exe`
- `Focused Moment Setup vX.Y.Z.exe`

推荐优先使用带版本号的文件进行测试或留档。

## 如果你要自己构建

启动开发环境：

```bash
pnpm tauri dev
```

导出调试版：

```bash
pnpm package:debug
```

导出正式版：

```bash
pnpm package:release
```

把已经构建好的当前版本同步到 GitHub Release：

```bash
pnpm release:github
```

如果你希望按固定顺序一口气完成：
`构建 -> 推送 main -> 打 tag -> 推送 tag -> GitHub Release`

```bash
pnpm release:ship
```

只清理根目录旧构建产物：

```bash
pnpm clean:artifacts
```
