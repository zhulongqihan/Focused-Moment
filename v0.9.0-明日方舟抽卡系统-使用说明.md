# Focused Moment v0.9.0 - 明日方舟抽卡养成系统

## 🎮 版本说明

本版本将原有的"量子仓鼠"宠物系统升级为完整的明日方舟风格抽卡养成系统，实现了番茄钟奖励与干员养成的完整闭环。

## 📦 安装文件

**可执行文件**: `Focused-Moment-v0.9.0-gacha-system.exe`

双击运行即可安装，安装后启动应用。

## ✨ 核心功能

### 1. 番茄钟计时系统
- 支持工作/休息模式切换
- 支持倒计时/正向计时两种模式
- Boss番茄钟机制（达到每日目标时触发）
- 作战委托挑战系统

### 2. 奖励系统（核心闭环）
完成番茄钟后自动获得奖励：
- **普通番茄钟**: 100合成玉 + 300龙门币 + 50经验值
- **Boss番茄钟**: 200合成玉 + 500龙门币 + 100经验值
- **挑战完成**: 额外 +50合成玉 + 200龙门币

### 3. 抽卡系统（寻访系统）
- **单次寻访**: 消耗600合成玉
- **十连寻访**: 消耗6000合成玉，保底至少一个5★或6★
- **概率机制**: 完全复刻明日方舟
  - 6★: 2% (保底后递增)
  - 5★: 8%
  - 4★: 50%
  - 3★: 40%
- **保底机制**: 
  - 50抽后6★概率开始递增
  - 99抽必出6★

### 4. 干员收藏系统
- 查看所有已获得的干员
- 按稀有度、职业筛选
- 按稀有度、等级、获得时间排序
- 重复干员自动提升潜能（最高6潜）

### 5. 干员养成系统
- **等级提升**: 消耗龙门币和经验值
  - Elite 0: 最高50级
  - Elite 1: 最高70级
  - Elite 2: 最高90级
- **精英化**: 消耗龙门币、经验值和芯片
  - 精英化后等级重置为1，等级上限提升
  - 需要达到当前等级上限才能精英化

## 💰 货币和资源说明

### 货币类型
1. **合成玉 (Orundum)** - 抽卡货币
   - 用途: 单抽(600) / 十连(6000)
   - 获取: 完成番茄钟工作会话

2. **龙门币 (LMD)** - 养成货币
   - 用途: 升级干员、精英化干员
   - 获取: 完成番茄钟工作会话

3. **经验值 (EXP)** - 养成资源
   - 用途: 升级干员、精英化干员
   - 获取: 完成番茄钟工作会话

4. **芯片 (Chips)** - 精英化材料
   - 用途: 精英化干员时需要
   - 获取: 完成番茄钟工作会话

## 🎯 完整游戏循环

```
完成番茄钟 → 获得奖励(合成玉/龙门币/经验值/芯片)
    ↓
使用合成玉抽卡 → 获得干员
    ↓
使用龙门币和经验值 → 升级干员
    ↓
使用芯片和资源 → 精英化干员
    ↓
继续完成番茄钟 → 获得更多资源...
```

## 🚀 快速开始

1. 安装并启动应用
2. 在"番茄战场"页面开始一个番茄钟
3. 完成后自动获得合成玉等奖励
4. 点击"🎲 寻访系统"进行抽卡
5. 点击"📋 干员收藏"查看和养成干员
6. 点击干员卡片进入详情页进行升级和精英化
